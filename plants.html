Result: When inspected in the browser the plants row breaks first.
Explanation: Eventhough we are using .col-sm and .col-md classes the columns by default would fit to the extra-large and large screens. But since we are using .col-md class it would shrink for the devices which are having screen width less than medium devices i.e,. at the breakpoint 767px screen width and less and remains intact for >=768px. In similar fashion, the letters row shrinks at screen width less than 576px.
